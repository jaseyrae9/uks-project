# uks-project

## Install requirements

$ pip install -r requirements.txt

## Initialize database

$ python manage.py makemigrations <br/>
$ python manage.py migrate

## Create Django admin user

$ python manage.py createsuperuser

## Run application

$ python manage.py runserver
